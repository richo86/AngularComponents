Usage Instructions

To use the slide-toggle component, import the SlideToggleComponent and add it to your template: 


        <app-slide-toggle [(checked)]="isToggled">Enable feature</app-slide-toggle>
        <p>Current state: {{ isToggled ? 'Enabled' : 'Disabled' }}</p>

The checked property supports two-way binding for reactive updates.

Component Details

  • HTML Structure: Custom slide toggle switch and label.
  • CSS Styling: Modern toggle styles, focus, and selection effects.
  • TypeScript Logic: Reactive toggle and event emission.

Easily customize the styles and behaviors to fit your needs.
