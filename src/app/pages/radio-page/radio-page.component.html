Usage Instructions

To use the radio component, import the RadioComponent and add it to your template: 


    <app-radio [(selectedOption)]="selectedOption"></app-radio>
    <p>Selected option: {{ selectedOption() }}</p>

The selectedOption property supports two-way binding for reactive updates.

Component Details

  • HTML Structure: Custom radio button group.
  • CSS Styling: Modern radio styles, focus, and selection effects.
  • TypeScript Logic: Reactive selection and event emission.

Easily customize the options, styles, and behaviors to fit your needs.
