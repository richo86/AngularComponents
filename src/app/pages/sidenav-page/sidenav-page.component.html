Sidenav Component Documentation

Welcome to the documentation page for the SidenavComponent. This guide explains how to integrate, customize, and use the responsive sidebar navigation—all with a sleek dark theme and dynamic content projection.

Overview

The SidenavComponent is built with Angular and combines a responsive sidebar with content projection via <ng-content>. When toggled, the sidebar moves between an expanded state (250px wide, showing icons, text, and a logo) and a collapsed state (35px wide, displaying only the icons). The main content, provided by the parent component, is automatically displaced to the right.

Usage Example

To use the SidenavComponent, first import it into your Angular module. Then wrap your page content within its tags. For instance: 

<app-sidenav>
  <div class="your-page-content">
    <h1>Your Page Title</h1>
    <p>Your content goes here...</p>
  </div>
</app-sidenav>

Customization

Fully adjust the sidebar’s appearance by editing the CSS. You can change the width values, colors, fonts, and transitions to suit your application’s aesthetics.

Toggle Functionality

The sidebar includes an internal toggle button that switches between the expanded and collapsed states. Additionally, you can control this behavior externally with Angular’s ViewChild if needed.
