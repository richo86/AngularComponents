Usage Instructions

To integrate this component into your Angular project, follow these steps:

  1. Add to Module: Ensure the CheckComponent is declared in your Angular module.
  2. Use in the Template: Insert the component using its selector. The component supports two-way binding on its model property. For example: 

    <app-check [label]="'Accept Terms'" [(model)]="isChecked"></app-check>
    <p>Status: {{ isChecked() ? 'Checked' : 'Unchecked' }}</p>

The two-way binding for the model property ensures that both the parent and child stay in sync with the checkbox state.

Component Code Details

Our custom checkbox component replaces the default browser checkbox with a modern, interactive design:

  • HTML Structure: Hides the native checkbox and uses a visually styled <span> element to display the check indicator.
  • CSS Styling: Features smooth transitions, color changes on check, hover, and focus states for enhanced interactivity and accessibility.
  • TypeScript Logic: Uses Angular’s reactive model API for streamlined state management.

You can easily customize the colors, sizes, and transitions to match the project needs.
